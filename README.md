## Visão Geral

Aplicação que simula a abertura de um biscoito da sorte chinês.

## Como Usar

Ao clicar na umagem do biscoito da sorte, ela faz uma animação de tremor. Após o clique, surge uma nova tela com a mensagem gerada.
